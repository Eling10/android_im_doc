消息管理

消息管理 ELChatManager

管理所有与聊天相关的逻辑，包括单聊和群聊消息的查询、发送、更新、删除的功能。

消息 Message

消息：IM 交互的实体,目前的消息体的类型有：文本消息、图片消息、文件消息、语音消息、视频消息、音频通话、视频通话。
